# Security Policy

## Supported Versions

|  Version  | Supported          |
| --------- | ------------------ |
| 3.1.x     | :white_check_mark: |
| < 3.1.x   | :x:                |

## Reporting a Vulnerability

If you think you have found a security issue, DO NOT open an issue. You MUST email your issue: security AT
spomky-labs.com.
